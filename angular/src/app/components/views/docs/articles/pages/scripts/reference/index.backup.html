Accessing Sprite References

You can access sprite references after the '// EDGTOKEN_CREATE' line as follows. 

class LevelScene_Title_Screen extends Phaser.Scene{{"{"}}

Sprite properties can be changed as follows.

  • .setPosition(x, y) - Set the position of the Sprite
  • .setSize(width, height) - Set the size of the Sprite
  • .setRotation(degrees) - Set the rotation of the Sprite

The following example shows how to use property changing methods. 

const mario = Sprite('mario-1.png');
mario.setPosition(100, 200);
merio.setSize(50, 50);

Sprite('cloud-1.png').setRotation(45);
